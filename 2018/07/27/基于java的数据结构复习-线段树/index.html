线段树(Segment Tree)

  • 线段树的每一个节点都是一个线段(区间)
  • 线段树不是完全二叉树
  • 线段树是平衡二叉树(最大深度与最小深度差值不超过1)
  • 如果区间有n个元素,线段树数组表示需要4n的空间

  • 我们的线段树不考虑添加元素，即区间固定，使用4n的静态空间即可
